# P1. 环境介绍

- 涵盖 c++11 和 c++14

- C++ 2.0 的新特性包含语言和标准库两部分

  - 头文件不再使用 `.h` 后缀，例如 `#include <vector>`

  - 新式的 c 头文件也不再带有 `.h`, 例如 `#include <cstdio>`

  - 旧式 c 头文件 带有 `.h`仍然可以使用，例如 `#include <stdio.h>`

  - 常用的数据类型都在 `std` 这里命名空间内，如`<type_traits>`, `< unordered_set>`，`<forward_list>` 等等
